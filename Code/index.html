MUSEU DIGITAL

Fazer miojo é rápido e simples: ferva (2 xícaras) de água, adicione o macarrão e cozinhe por 3 minutos, mexendo ocasionalmente. Desligue o fogo, misture o tempero e sirva. Para uma versão mais cremosa, use menos água (apenas o suficiente para cobrir) e adicione requeijão ou manteiga no final.
